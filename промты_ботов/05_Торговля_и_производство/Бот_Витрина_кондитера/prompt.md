# Промт для создания Telegram-бота «Витрина кондитера»

Ты — senior Python-разработчик, специализирующийся на создании Telegram-ботов с использованием библиотеки aiogram 3.x. Твоя задача — написать полноценного, готового к развёртыванию на сервере Telegram-бота «Витрина кондитера».

**Контекст проекта:**
Этот бот продаётся домашним кондитерам и небольшим кондитерским. Цель — показывать торты и десерты с фото и ценами, собирать заявки на заказ с указанием даты и пожеланий. Бот заменяет Instagram-витрину и упрощает приём заказов.

**Функциональные требования:**

1. **Стартовое сообщение и меню:**
   - При команде `/start` бот отправляет приветственное сообщение с названием кондитерской.
   - Под сообщением — клавиатура (InlineKeyboardMarkup):
     - 🎂 Каталог тортов
     - 🧁 Десерты и капкейки
     - 📝 Мой заказ
     - 📞 Контакты

2. **Обработка кнопок и сценарии:**
   - **Каталог тортов:** Показывает список тортов с фото, описанием, весом и ценой. При выборе — детальная информация и кнопка «Заказать».
   - **Десерты и капкейки:** Аналогично, отдельная категория товаров.
   - **Мой заказ:** Показывает текущий статус заказа (если есть) или кнопку «Оформить новый».
   - **Контакты:** Телефон, адрес самовывоза, режим работы, ссылка на Instagram.

3. **Администрирование:**
   - Команда `/add_product [name] [category] [price] [description] [photo]` — добавить товар в каталог.
   - Команда `/list_orders` — показать все новые заказы.
   - Команда `/update_order_status [order_id] [status]` — изменить статус заказа.
   - Команда `/broadcast [текст]` — рассылка всем подписчикам.

4. **Технические детали:**
   - **Стек:** Python 3.10+, aiogram 3.x, python-dotenv, aiosqlite, aiofiles, asyncio.
   - **Структура проекта:**
```
/bot
├── main.py
├── config.py
├── handlers/
│   ├── user.py
│   └── admin.py
├── keyboards/
│   └── main_menu.py
├── data/
│   └── products.json
├── photos/ # папка для фото товаров
├── database.py
├── .env
└── requirements.txt
```
   - **Конфигурация (.env):**
```
BOT_TOKEN=123456:ABCdef...
ADMIN_ID=123456789
CONFECTIONERY_NAME=«Сладкая мечта»
CONTACT_PHONE=+79001234567
ADDRESS=г. Москва, ул. Сладкая, 5
INSTAGRAM=https://instagram.com/...
```
   - **Хранение данных:** SQLite с таблицами:
     - `products` (id, name, category, price, description, photo_path, is_active)
     - `orders` (id, user_id, username, phone, product_id, order_date, delivery_date, comments, status)
     - `users` (user_id, username, first_name, last_seen)

5. **Обработка ошибок и безопасность:**
   - Все запросы к БД с обработкой исключений.
   - При ошибке — пользователю вежливое сообщение, админу — лог в ЛС.
   - Рассылка с проверкой на блокировку бота.

6. **Деплой:**
   - Код готов к запуску через Docker или systemd.
   - Включить Dockerfile, docker-compose.yml, README.md.
