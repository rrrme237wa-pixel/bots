Ты — senior Python-разработчик, специализирующийся на создании Telegram-ботов с использованием библиотеки aiogram 3.x. Твоя задача — написать полноценного, готового к развёртыванию на сервере Telegram-бота «Тайный Санта».

**Контекст проекта:**
Этот бот продаётся как готовое решение для организаторов корпоративов. Цель — распределение участников и вишлисты. Монетизация — продажа бота бизнесу за фиксированную плату или подписку.

**Функциональные требования:**

1. **Стартовое сообщение и меню:**
   - При команде `/start` бот отправляет приветственное сообщение с названием сервиса и описанием возможностей.
   - Главная клавиатура (ReplyKeyboardMarkup):
🎅 Участвовать
     - 🎁 Мой подопечный
     - 📝 Вишлист
     - 📊 Участники

2. **Обработка команд:**
Регистрация, авто-распределение, вишлисты.

3. **Администрирование:**
   - Команда `/admin` — статистика, заявки, рассылка.
   - Команда `/broadcast [текст]` — рассылка всем пользователям.
   - Команда `/export` — выгрузка данных в CSV.

4. **Технические детали:**
   - **Стек:** Python 3.10+, aiogram 3.x, aiosqlite, apscheduler.
   - **Структура:** main.py, config.py, database.py, handlers/, keyboards/, utils/, .env, requirements.txt
   - **.env:** BOT_TOKEN, ADMIN_ID, GAME_NAME, EXCHANGE_DATE

5. **Хранение данных:** SQLite с таблицами users, requests, participants (id, game_id, user_id, wishlist, assigned_to)

6. **Безопасность:** Таймауты, валидация, защита админских команд.

7. **Деплой:** Docker-ready код с инструкцией.

**Формат ответа:** Полный код всех файлов с комментариями.

Напиши код бота согласно этому заданию.
