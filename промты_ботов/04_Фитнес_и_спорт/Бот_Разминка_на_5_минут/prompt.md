# Промт для создания Telegram-бота «Разминка на 5 минут»

Ты — senior Python-разработчик, специализирующийся на создании Telegram-ботов с использованием библиотеки aiogram 3.x. Твоя задача — написать полноценного, готового к развёртыванию на сервере Telegram-бота «Разминка на 5 минут».

**Контекст проекта:**
Этот бот продаётся фитнес-клубам, тренерам и ЗОЖ-блогерам как бонус для клиентов или монетизируется через платную подписку. Цель — предоставлять пользователям случайные короткие комплексы упражнений (разминка, растяжка, зарядка) с GIF-анимациями. Бот помогает клиентам оставаться активными между тренировками.

**Функциональные требования:**

1. **Стартовое сообщение и меню:**
   - При команде `/start` бот отправляет приветственное сообщение с описанием возможностей.
   - Под сообщением — клавиатура (InlineKeyboardMarkup):
     - 🔥 Случайная разминка
     - 📋 Комплексы по категориям
     - ⏱️ Таймер тренировки
     - 📊 Моя статистика
     - 💎 Премиум-комплексы

2. **Обработка кнопок и сценарии:**
   - **Случайная разминка:** Бот выбирает случайный комплекс из 5–7 упражнений. Каждое упражнение отправляется отдельным сообщением с GIF-анимацией, названием, описанием техники и количеством повторений. В конце — итоговое сообщение с общим временем выполнения.
   - **Комплексы по категориям:** Пользователь выбирает категорию (Утренняя зарядка, Разминка перед тренировкой, Растяжка вечером, Офисная гимнастика), бот выдаёт соответствующий комплекс.
   - **Таймер тренировки:** Запускает таймер на время комплекса (например, 5 минут) с обратным отсчётом.
   - **Моя статистика:** Показывает количество выполненных комплексов, общее время тренировок, серию дней подряд.
   - **Премиум-комплексы:** Если пользователь оформил подписку (флаг в БД), доступны расширенные программы.

3. **Администрирование:**
   - Команда `/add_exercise [name] [category] [gif_file] [description] [reps]` — добавить упражнение в базу.
   - Команда `/add_routine [name] [exercises_list]` — создать готовый комплекс.
   - Команда `/list_exercises` — показать все упражнения.
   - Команда `/broadcast [текст]` — рассылка всем пользователям.
   - Команда `/set_premium [user_id]` — выдать премиум-доступ пользователю.

4. **Технические детали:**
   - **Стек:** Python 3.10+, aiogram 3.x, python-dotenv, aiosqlite, aiofiles, asyncio.
   - **Структура проекта:**
```
/bot
├── main.py
├── config.py
├── handlers/
│   ├── user.py
│   └── admin.py
├── keyboards/
│   └── main_menu.py
├── data/
│   └── exercises.json
├── gifs/ # папка для GIF-анимаций
├── database.py
├── .env
└── requirements.txt
```
   - **Конфигурация (.env):**
```
BOT_TOKEN=123456:ABCdef...
ADMIN_ID=123456789
BOT_NAME=FitMinute
PREMIUM_PRICE=299
TIMEZONE=Europe/Moscow
```
   - **Хранение данных:** SQLite с таблицами:
     - `exercises` (id, name, category, gif_path, description, reps, duration_sec)
     - `routines` (id, name, exercises_json, is_premium, created_at)
     - `user_stats` (user_id, total_workouts, total_minutes, streak_days, last_workout, is_premium)
     - `users` (user_id, username, first_name, last_seen)

5. **Обработка ошибок и безопасность:**
   - Кэширование GIF для быстрой загрузки.
   - Обработка случая, когда GIF-файл не найден (замена на текстовое описание).
   - Логирование активности пользователей для статистики.

6. **Деплой:**
   - Код готов к запуску через Docker или systemd.
   - Включить Dockerfile, docker-compose.yml, README.md.
