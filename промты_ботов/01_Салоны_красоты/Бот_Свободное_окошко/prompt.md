# Промт для создания Telegram-бота «Свободное окошко»

Ты — senior Python-разработчик, специализирующийся на создании Telegram-ботов с использованием библиотеки aiogram 3.x. Твоя задача — написать полноценного, готового к развёртыванию на сервере Telegram-бота «Свободное окошко».

## Контекст проекта

Этот бот продаётся как готовое цифровое решение для частных мастеров сферы красоты (маникюр, брови, парикмахеры, косметологи). Цель — позволить мастеру быстро уведомить подписчиков о внезапно освободившемся времени для записи со скидкой, чтобы заполнить «окна» в расписании и не терять доход.

**Монетизация:** Продажа готового решения мастерам за 3000–5000 рублей, либо ежемесячная подписка 500 рублей за поддержку и обновления.

## Функциональные требования

### 1. Стартовое сообщение и меню

- При команде `/start` бот отправляет приветственное сообщение с именем мастера.
- Под сообщением — клавиатура InlineKeyboardMarkup:
  - 🔔 Получать уведомления об окнах
  - 📅 Актуальные окна
  - 📞 Записаться
  - ℹ️ О мастере

### 2. Обработка кнопок

#### Получать уведомления об окнах
- Клиент подписывается на рассылку уведомлений о свободных окнах.
- Можно выбрать предпочтительные дни недели и время (утро/день/вечер).
- Кнопка «Отписаться» для отключения уведомлений.

#### Актуальные окна
- Бот показывает список текущих свободных окон со скидкой.
- Для каждого окна: дата, время, размер скидки (например: −20%).
- Кнопка «Записаться на это время» рядом с каждым окном.

#### Записаться
- Быстрая запись на выбранное окно.
- После записи окно автоматически удаляется из списка доступных.
- Мастер получает уведомление о записи.

#### О мастере
- Информация о мастере, прайс-лист, контакты.

### 3. Администрирование (для мастера)

Для владельца бота (ID админа задаётся в `.env`) доступны команды:

- `/admin` — статистика:
  - Количество подписчиков
  - Количество отправленных уведомлений
  - Количество записей через бота

- `/add_window [дата] [время] [скидка]` — добавить свободное окно:
  - Пример: `/add_window 2024-03-15 14:00 20`
  - Бот создаёт запись и предлагает отправить уведомление подписчикам.

- `/send_notification [текст]` — отправить уведомление о новом окне всем подписчикам.
  - Автоматически добавляет информацию о скидке.
  - Кнопка «Записаться» в сообщении.

- `/remove_window [id]` — удалить окно из списка.

- `/list_windows` — показать все активные окна.

- `/broadcast [текст]` — общая рассылка подписчикам.

### 5. Хранение данных

**SQLite база данных:**

Таблица `users`:
- `user_id` INTEGER PRIMARY KEY
- `username` TEXT
- `first_name` TEXT
- `phone` TEXT
- `subscribed_to_slots` BOOLEAN DEFAULT 0
- `preferred_days` TEXT (JSON массив)
- `preferred_time` TEXT (morning/afternoon/evening)
- `first_seen` TIMESTAMP

Таблица `available_slots`:
- `id` INTEGER PRIMARY KEY AUTOINCREMENT
- `slot_date` DATE
- `slot_time` TIME
- `discount_percent` INTEGER
- `is_booked` BOOLEAN DEFAULT 0
- `booked_by_user_id` INTEGER
- `created_at` TIMESTAMP
- `expires_at` DATETIME

Таблица `notifications_log`:
- `id` INTEGER PRIMARY KEY AUTOINCREMENT
- `sent_at` TIMESTAMP
- `recipients_count` INTEGER
- `message_text` TEXT

При каждом `/start` обновлять запись в `users`.

### 6. Логика работы

**Добавление окна:**
1. Мастер вводит команду `/add_window` с параметрами.
2. Бот проверяет корректность даты/времени.
3. Сохраняет окно в базу данных.
4. Предлагает отправить уведомление подписчикам.

**Уведомление подписчиков:**
1. Бот выбирает всех пользователей с `subscribed_to_slots = 1`.
2. Фильтрует по предпочтениям (дни недели, время суток).
3. Отправляет персонализированное сообщение:
   ```
   🔥 Свободное окошко!
   
   {дата} в {время}
   Скидка {discount}%!
   
   Успейте записаться — предложение ограничено!
   ```
4. Добавляет кнопку «Записаться» с callback_data для быстрого действия.

**Запись на окно:**
1. Клиент нажимает «Записаться».
2. Бот помечает окно как забронированное.
3. Отправляет подтверждение клиенту.
4. Уведомляет мастера о новой записи.

### 7. Обработка ошибок и безопасность

- Таймауты на запросы (10 сек).
- Рассылка с задержкой 0.05 сек между сообщениями.
- Обработка `TelegramForbiddenError` — отписка пользователя.
- Автоматическое удаление просроченных окон (по истечении `expires_at`).
- Защита от двойной записи на одно окно (блокировка при одновременном нажатии).

### 8. Деплой

**Dockerfile:**
```dockerfile
FROM python:3.10-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
CMD ["python", "main.py"]
```

**docker-compose.yml:**
```yaml
version: '3.8'
services:
  bot:
    build: .
    restart: unless-stopped
    env_file: .env
    volumes:
      - ./data:/app/data
```

**systemd сервис:**
```ini
[Unit]
Description=Free Slot Bot
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/free-slot-bot
ExecStart=/usr/bin/python3 main.py
Restart=always

[Install]
WantedBy=multi-user.target
```

**Инструкция по развёртыванию на VPS:**

1. Скопировать файлы проекта на сервер
2. Создать файл `.env` с необходимыми переменными
3. Запустить через Docker: `docker-compose up -d`
4. Или через systemd:
   ```bash
   sudo cp free-slot-bot.service /etc/systemd/system/
   sudo systemctl daemon-reload
   sudo systemctl enable free-slot-bot
   sudo systemctl start free-slot-bot
   ```

5. Проверить логи: `journalctl -u free-slot-bot -f`
